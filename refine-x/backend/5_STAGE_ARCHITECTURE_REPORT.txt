6. Updated CleanedDataset to include ai_classification_flags and ai_formulas

Execution Order (FINAL):
  1. GLOBAL rules (Session 1) → Normalizes column names, removes BOM, etc.
  2. STRUCT rules (Session 2) → Fixes layout, header alignment
  3. classify_and_assign() → SINGLE GPT call returns HTYPE + formulas
  4. HtypeDetector (Fallback) → Runs for all columns
  5. Tiered Merge:
     • confidence >= 0.85 → use AI
     • confidence 0.60-0.84 → check agreement, disagree → PendingReview
     • confidence < 0.60 → always PendingReview
  6. Personal Identity Rules → Uses merged htype_map
  7. Date/Time Rules → Uses merged htype_map
  8. ... (all remaining sessions use merged htype_map)

Code Path (from process_csv.py):
```python
# Tiered Confidence Thresholds
AI_CONFIDENCE_HIGH = 0.85     # Auto-apply AI classification
AI_CONFIDENCE_LOW = 0.60      # Below this → always PendingReview

# Single GPT call for HTYPE + Formulas
ai_result = classify_and_assign(
    columns=df.columns.tolist(),
    sample_rows=sample_rows,
    filename=job.filename,
)
ai_classification = ai_result.get("columns", {})

# HtypeDetector (Fallback)
htype_detector = HtypeDetector()
htype_report = htype_detector.get_detection_report(df)
htype_map = htype_report["htype_map"]

# Tiered Merge
for col, ai_data in ai_classification.items():
    ai_htype = ai_data.get("htype", "HTYPE-000")
    ai_confidence = ai_data.get("confidence", 0.0)
    detector_htype = htype_map[col]
    
    if ai_confidence >= AI_CONFIDENCE_HIGH:
        # High confidence: use AI directly
        htype_map[col] = ai_htype
        
    elif ai_confidence >= AI_CONFIDENCE_LOW:
        # Medium confidence: check agreement
        if ai_htype == detector_htype:
            htype_map[col] = ai_htype
        else:
            # Conflict → PendingReview
            ai_flags.append({
                "flag_type": "HTYPE_CLASSIFICATION_CONFLICT",
                "column": col,
                "ai_htype": ai_htype,
                "ai_confidence": ai_confidence,
                "detector_htype": detector_htype,
                "message": f"Please confirm column type for '{col}'",
                "pending_review": True,
            })
    else:
        # Low confidence → always PendingReview
        ai_flags.append({
            "flag_type": "HTYPE_CLASSIFICATION_LOW_CONFIDENCE",
            ...
            "pending_review": True,
        })
```

SINGLE SOURCE OF TRUTH: There is now exactly ONE htype_map that the
entire pipeline uses. AI classification is primary for high confidence.
HtypeDetector provides fallback for low confidence. Conflicts require
user confirmation via PendingReview before cleaning proceeds.
